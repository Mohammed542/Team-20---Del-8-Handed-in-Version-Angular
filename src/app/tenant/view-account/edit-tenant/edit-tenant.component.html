Hint: Edit Profile

This screen allows you to edit your account information.

Your Name, Surname, and Cellphone Number can be edited.

You cannot edit your Email Address or your ID Number as it is used throughout the system for various Landlord functionality.

All fields demarcated with an asterisk (*) have to have a value for the account information to be allowed to update.

You can select the "Cancel" button located at the bottom of the editable fields to return to your profile page without making changes.

Once the required fields are populated, you can select the "Confirm Changes" button to update your edited account information.
