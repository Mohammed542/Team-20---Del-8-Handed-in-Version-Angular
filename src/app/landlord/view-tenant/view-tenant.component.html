Hint: View Tenants

This screen allows you to view all Tenants registered onto the system.

Located at the top right of the table there is a "Search Tenants" search bar. Enter the name of the Tenant to filter the list with matching criteria.

NOTE: When searching for a Tenant using the search bar, ensure that the name of the Tenant you are searching for is Case Sensitive!

The Tenant's full name, their residing building, their residing unit number, and the status of their account can be seen in the table.

You can select the various headings at the top of the table (Tenant, Building, etc.) to sort the table using the selected criteria.

Select the three dots located underneath the "Options" column to find various options that can be selected for the chosen Tenant.

You can select the "View Tenant Profile" in the options menu to display the Name, Surname, Email Address, Cellphone Number, and the ID Number of the chosen Tenant.

You can select the "Deactivate Tenant Account" button to deactivate the Tenant's account within the system.

WARNING: Once deactivation is confirmed, the account will be fully deactivated and the Tenant will no longer have access to the system.

You can select the "Items per page" dropdown to reveal a list of numbers. Select the number of how many Tenants you would like to see on the table.

Alternatively, you can select the various arrows located next to the "Items per page" dropdown to skim through pages of Tenants registered on the system.

You can select the "Return to Homepage" button located at the bottom to return to the home screen.
