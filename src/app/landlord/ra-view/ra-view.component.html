This screen allows you to select what document you would like to view.

There are four buttons which take you to the necessary page that allows you to view the required documents.

Select the "View Rental Agreement Document" button to navigate to the page that displays a table of all Rental Agreement documents on the system.

Select the "View Tenant ID Document" button to navigate to the page that displays a table of all Tenant ID documents on the system.

Select the "View Company's CIPC Document" button to navigate to the page that displays a table of all company's CIPC documents on the system.

Select the "View Company's ID Document" button to navigate to the page that displays a table of all company's ID documents on the system.

You can select the "Return to Homepage" button located at the bottom to return to the home screen.
