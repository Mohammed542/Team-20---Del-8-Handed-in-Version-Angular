Hint: View Company's Companies and Intellectual Property Commission Documents

This screen allows you to view all Company's CIPC documents available on the system.

The Company's name, their residing building, their residing unit number, the start and end date of the company's Rental Agreement, and the status of their Rental Agreement can be seen in the table.

You can select the various headings at the top of the table (Company Name, Building, etc.) to sort the table using the selected criteria.

Select the three dots located underneath the "Options" column to find various options that can be selected for the chosen company's CIPC document.

You can select the "View Company's CIPC Document" in the options menu to display the Company's CIPC document that has been uploaded.

You can select the "Items per page" dropdown to reveal a list of numbers. Select the number of how many companies you would like to see on the table.

Alternatively, you can select the various arrows located next to the "Items per page" dropdown to skim through pages of companies registered on the system.

You can select the "Return to Homepage" button located at the bottom to return to the home screen.
