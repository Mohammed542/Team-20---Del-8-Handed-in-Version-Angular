Hint: View Tenant's Identification Documents

This screen allows you to view all Tenant's Identification documents available on the system.

The Tenant's name, their residing building, their residing unit number, the start and end date of their Rental Agreement, and the status of their Rental Agreement can be seen in the table.

You can select the various headings at the top of the table (Company Name, Building, etc.) to sort the table using the selected criteria.

Select the three dots located underneath the "Options" column to find various options that can be selected for the chosen Tenant's ID document.

You can select the "View Tenant ID Document" in the options menu to display the Tenants's ID document that has been uploaded.

You can select the "Items per page" dropdown to reveal a list of numbers. Select the number of how many Tenant ID Documents you would like to see on the table.

Alternatively, you can select the various arrows located next to the "Items per page" dropdown to skim through pages of tenant's ID documents uploaded on the system.

You can select the "Return to Homepage" button located at the bottom to return to the home screen.
