Hint: View Landlords

This screen allows you to view all Landlords registered onto the system.

Located at the top right of the table there is a "Search Landlords" search bar. Enter the name of the Landlord to filter the list with matching criteria.

The Landlord's full name, email address, and cellphone number can be seen in the table.

You can select the various headings at the top of the table (Landlord, Cellphone Number, etc.) to sort the table using the selected criteria.

Select the three dots located underneath the "Options" column to find various options that can be selected for the chosen Landlord.

You can select the "View Landlord Profile" in the options menu to display the Name, Surname, Email Address, Cellphone Number, and the ID Number of the chosen Tenant.

You can select the "Items per page" dropdown to reveal a list of numbers. Select the number of how many Landlords you would like to see on the table.

Alternatively, you can select the various arrows located next to the "Items per page" dropdown to skim through pages of Landlords registered on the system.

Select the "Create Landlord Account" button to navigate to a page that allows you to fill in the necessary details to create a Landlord account.

You can select the "Return to Homepage" button located at the bottom to return to the home screen.
