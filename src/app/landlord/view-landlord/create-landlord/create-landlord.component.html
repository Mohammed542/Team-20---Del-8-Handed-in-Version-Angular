Hint: Create Landlord Account

This screen allows you to create a Landlord account using the information entered in the required fields.

There are 5 fields that need to be filled out in order to create a Landlord account.

The first field to enter is the First Name of the person for whomst you want to create a Landlord account for.

The second field to enter is the Surname of the person for whomst you want to create a Landlord account for.

The third field to enter is the cellphone number of the person for whomst you want to create a Landlord account for.

The fourth field to enter is the ID number of the person for whomst you want to create a Landlord account for.

The fifth field to enter is the Email Address of the person for whomst you want to create a Landlord account for.

WARNING: Make sure the email address you entered is 100% correct as an Email will be sent to the address containing information allowing the person to login to the system.

You can select the "Cancel" button located at the bottom to return to the list of Landlords without saving any information entered.

You can select the "Create" button located at the bottom to create a Landlord account using the entered details.
